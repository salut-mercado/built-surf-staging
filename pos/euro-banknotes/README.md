# Euro banknote presets

These WebP files are 256 px-wide renders of CC0 SVG artwork by `frankes`,
downloaded from Wikimedia Commons and used by the POS cash-payment presets.

- [5-Euro.svg](https://commons.wikimedia.org/wiki/File:5-Euro.svg)
- [10-Euro.svg](https://commons.wikimedia.org/wiki/File:10-Euro.svg)
- [20-Euro.svg](https://commons.wikimedia.org/wiki/File:20-Euro.svg)
- [50-Euro.svg](https://commons.wikimedia.org/wiki/File:50-Euro.svg)
- [100-Euro.svg](https://commons.wikimedia.org/wiki/File:100-Euro.svg)

Wikimedia identifies each source as CC0 / public-domain dedication. Banknote
reproduction rules may still apply independently of the artwork license.
